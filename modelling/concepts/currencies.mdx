When you create a variable with type `currency`,  you must specify which currency.

By default, Pluvo uses the currency you have selected in your accounting software. If you do not have an accounting system connected, you may need to manually add currencies in your organization settings before creating any currency forecasts (see: [#add-or-remove-currencies](#add-or-remove-currencies "mention")).

To change the currency of a currency variable, click the variable type icon and select currency. Then you can pick your desired currency from the dropdown.
